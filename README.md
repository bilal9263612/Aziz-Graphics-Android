# Aziz Graphics: Minimalist HTML File Manager for Android

**Aziz Graphics** is a lightweight, minimalist Android application designed to embed and manage local HTML files within its own interface. It provides a clean, dark-themed environment for users to organize and quickly access their HTML-based tools or content, ensuring they remain accessible even if the original file is moved or deleted from the device's storage.

The application is built with a focus on performance, minimal resource usage, and a unique, modern UI aesthetic featuring glassy elements and a pure black background.
